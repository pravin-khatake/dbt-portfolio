# dbt Analytics Portfolio

This repository demonstrates dbt-style analytics engineering practices:
- Staging models
- Fact and dimension tables
- Business-focused SQL transformations

## Project Goal
Build clean, reusable data models to support BI dashboards and decision-making.

## Folder Structure
- models/staging: Cleaned source data
- models/marts: Fact and dimension tables
- tests: Data quality checks
- analysis: Ad-hoc analysis
